# The Station - A rails news website

> This a captsone project for the Microverse Program, all the requirements and design concepts are included.

![screenshot](https://i.imgur.com/ot4c6iT.png)

## Built With

- Ruby v2.7.0
- Ruby on Rails v6.0
- CSS Flexbox

## Live Demo

[Hosted on Heroku](#)

### Setup

Clone this project:
```
git clone git@github.com:idgm5/lifestyle.git
```

Install gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
```

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    rpsec --format documentation
```
